HOW IT WORKS
============

1. Push code to GitHub main branch
2. GitHub Actions automatically triggers
3. Workflow:
   - Checks out code
   - Sets up JDK 17
   - Grants gradlew permissions
   - Builds APK with: ./gradlew assembleDebug
   - Uploads APK as artifact
4. Download APK from GitHub Actions tab
5. Install on device: adb install app-debug.apk

WORKFLOW TRIGGERS
=================

✅ Push to main branch
✅ Pull requests to main
✅ Manual trigger (workflow_dispatch)

BUILD TIME
==========

First build: 5-10 minutes
Subsequent builds: 2-3 minutes
(Gradle caches dependencies)

APK LOCATION
============

Built at: build/outputs/apk/debug/app-debug.apk
Downloaded from: GitHub Actions → Artifacts → DataWatchdog-APK

SETUP STEPS
===========

1. Create GitHub repository
2. Push code to main branch
3. Go to Settings → Actions → General
4. Select "Allow all actions and reusable workflows"
5. Click Save
6. Go to Actions tab
7. Wait for build to complete
8. Download APK from artifacts
