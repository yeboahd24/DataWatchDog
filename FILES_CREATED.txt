Database Layer (db/)
--------------------
src/main/kotlin/com/datawatchdog/db/AppDatabase.kt
  - Room database singleton
  - 3 tables: data_usage, bundle_info, drain_alerts
  - Thread-safe instance creation

src/main/kotlin/com/datawatchdog/db/DataUsageEntity.kt
  - DataUsageEntity: per-app usage records
  - BundleEntity: current bundle information
  - DrainAlertEntity: drain detection alerts

src/main/kotlin/com/datawatchdog/db/DataUsageDao.kt
  - DataUsageDao: usage data access
  - BundleDao: bundle data access
  - DrainAlertDao: alert data access

Utilities (util/)
-----------------
src/main/kotlin/com/datawatchdog/util/DataUsageTracker.kt
  - NetworkStatsManager wrapper
  - Per-app mobile and WiFi tracking
  - UID to package name conversion
  - Byte to MB conversion

src/main/kotlin/com/datawatchdog/util/SmsParser.kt
  - SMS message parsing
  - Provider detection (MTN, Vodafone, AirtelTigo)
  - Expiry date extraction with regex
  - Data amount parsing

src/main/kotlin/com/datawatchdog/util/DrainDetector.kt
  - Drain detection algorithm
  - Rolling window of last 10 readings
  - 2MB/minute threshold
  - Drain rate calculation

src/main/kotlin/com/datawatchdog/util/BundlePredictor.kt
  - Bundle exhaustion prediction
  - Average usage calculation
  - Linear extrapolation
  - Time-to-exhaustion calculation

src/main/kotlin/com/datawatchdog/service/DataMonitorService.kt
  - Foreground service

  - Data usage monitoring
  - Drain detection

src/main/kotlin/com/datawatchdog/receiver/SmsReceiver.kt
